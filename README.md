Uma API robusta e escalável desenvolvida para fornecer informações detalhadas sobre a composição nutricional de alimentos consumidos no Brasil. O projeto utiliza dados estruturados para facilitar a integração com aplicativos de saúde, fitness e nutrição.

---

## 🚀 Demonstração Online
A API está hospedada no Render e integrada ao banco de dados Supabase:
- **URL Base:** [https://tabelanutricionalbrasileira.onrender.com](https://tabelanutricionalbrasileira.onrender.com)
- **Documentação Interativa (Swagger):** [Acesse aqui](https://tabelanutricionalbrasileira.onrender.com/swagger-ui.html)

---

## 🛠️ Tecnologias e Ferramentas
- **Backend:** Java 21 com Spring Boot 3.4.3
- **Persistência de Dados:** Spring Data JPA & Hibernate
- **Banco de Dados:** PostgreSQL (Cloud via Supabase)
- **Containerização:** Docker & Docker Compose
- **Documentação:** SpringDoc OpenAPI (Swagger UI)
- **Auxiliares:** Lombok, MapStruct (DTO Mapping) e OpenCSV (Processamento de dados)
- **Deploy:** Render (Estratégia de Build via Dockerfile)
